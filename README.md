# install golang 
```shell
wget https://go.dev/dl/go1.20.6.linux-amd64.tar.gz && rm -rf /usr/local/go && tar -C /usr/local -xzf go1.20.6.linux-amd64.tar.gz

```

env setting:
```shell
export PATH=$PATH:/usr/local/go/bin
```

check version:
```shell
go version
```
